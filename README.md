# Divine Foods – Sales Performance Analysis

## Overview
This project analyses sales performance data for Divine Foods to identify trends, key performance drivers, and opportunities to improve business decision-making. The analysis focuses on transforming raw sales data into clear, actionable insights suitable for non-technical stakeholders.

---

## Objectives
- Analyse overall sales performance and trends
- Identify top- and under-performing products
- Understand revenue contribution by category and time period
- Present insights in a clear, stakeholder-friendly format

---

## Tools Used
- Microsoft Excel – data cleaning, analysis, visualisation
- PowerPoint – insight storytelling and business presentation

---

## Key Deliverables
- Cleaned and analysed sales dataset
- Analytical report summarising findings
- Executive-style presentation highlighting insights and recommendations

---

## Key Insights
- Identified sales trends and performance patterns across products
- Highlighted products contributing most to overall revenue
- Flagged areas of underperformance requiring further investigation
